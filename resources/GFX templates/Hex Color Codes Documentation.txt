GRADIENT COLOR REFERENCE (FOR PHOTOSHOP)

This document defines the standard gradient overlay color presets used for
map effects and UI visuals.

All values are HEX color codes and correspond to gradient location points
inside Photoshop’s Gradient Editor.

IMPORTANT
"Location" refers to the percentage position in the gradient (0–100).
Colors should be entered exactly as listed.
Do not manually adjust colors unless creating a new documented preset.

STANDARD GRADIENT (YELLOW -> BROWN)

Description:
Default gradient used for general overlays and base visual effects.

Color Stops:
Location 0 : #6A350F (dark brown base)
Location 100 : #E4C65F (warm yellow highlight)

MAP MODE - "..._selected"

Description:
Used when a map element is actively selected or highlighted.

Color Stops:
Location 0 : #FFFFFF (bright highlight)
Location 50 : #FFDEAF (warm mid-tone)
Location 100 : #907657 (muted brown shadow)

MAP MODE - "..._deselected"

Description:
Used for inactive or non-selected map elements.

Color Stops:
Location 0 : #DFDBC6 (desaturated light tone)
Location 100 : #46453E (neutral dark gray-brown)

PHOTOSHOP SETUP GUIDELINES

To recreate gradients:

Open Layer Style -> Gradient Overlay.
Open the Gradient Editor.
Add color stops at the listed locations.
Enter HEX values exactly as written.
Save gradients as presets if reused frequently.

Recommended baseline settings:
Blend Mode : Normal
Opacity : 100% (adjust per asset)
Scale : 100%

CONSISTENCY RULES
Always reuse documented gradients instead of recreating by eye.
Do not modify existing presets unless updating documentation.
